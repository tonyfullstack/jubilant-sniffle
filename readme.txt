to get your bot working, you need to navigate to .env and
add your bot token after TOKEN=

for instance: TOKEN=flmsdfr0694206969...

You will also need the following npm modules

- discord.js
- ytdl-core
- ffmpeg-static
- @discord.js/opus
- dotenv

You can run:

npm i discord.js ytdl-core ffmpeg-static @discord.js/opus dotenv

to install all of them.
